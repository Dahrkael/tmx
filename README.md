# Tmx

A library for parsing the **[Tiled Map Editor](http://www.mapeditor.org/)** file format.

## Limitations

* Support only for JSON format

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tmx'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install tmx
```
